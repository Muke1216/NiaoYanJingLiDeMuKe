# 🧸 木可的毛绒收藏工具
> 一款专为宝可梦毛绒玩具藏家设计的 PWA 收藏管理工具，可直接添加到手机桌面使用，无需下载 App。

## ✨ 核心功能
### 五大模块

| 模块 | 说明 |
|------|------|
| 🧸 **收藏** | 记录已入手的毛绒，支持图片、价格、品牌、系列、标签、备注等完整信息 |
| 📖 **图鉴** | 毛绒图鉴库，可从收藏一键同步，也可独立新增条目，支持独立照片 |
| 📦 **在途** | 记录已付款、运输中的毛绒，到货后一键移入收藏并同步图鉴 |
| 🎁 **周边** | 独立管理周边商品（挂件、卡牌、文具等），独立标签库与分组 |
| 📤 **分享** | 记录已分享或赠出的藏品 |

> 在途、周边、分享模块默认隐藏，在设置中按需开启。
---
### 收藏管理
- 每件藏品支持上传 **最多 3 张图片**（主图 + 补充图）
- 信息字段：名称、品牌、系列、买入价格（支持商品/运费/其他明细拆分）、持有数量、标签、备注
- 输入品牌后，系列栏自动联想该品牌的历史系列
- 分组管理：自由创建分组，支持按分组筛选，分组标签旁实时显示物品数量
- 多选操作：批量删除、批量移动到其他模块、批量设置分组

### 图鉴
- 从收藏详情点击「加入图鉴」一键同步，已在图鉴内时给出提示
- 支持独立新增图鉴条目（不依赖收藏数据）
- 图鉴可上传独立照片，与收藏图片互不干扰
- 删除图鉴条目不影响收藏数据，删除收藏数据图鉴仍保留

### 标签系统
- 每件藏品可添加多个自定义标签，支持点击标签快速筛选
- 历史标签自动记录，下次输入时一键快速添加
- 周边模块拥有独立标签库，与收藏互不干扰

### 备份与恢复
- 一键导出完整备份（JSON 格式，含所有图片数据）
- 支持按模块、按分组选择性导出
- 导入支持「完全替换」或「合并追加」两种模式

### 安全模式
- 设置中开启「🔒 安全模式」后，所有删除、图片修改、批量移动操作均需二次确认，防止误操作

## 🛠 技术说明

- **纯静态单文件**：全部逻辑写在一个 `index.html` 中，无需服务端，部署到任何静态托管（GitHub Pages 等）即可使用
- **PWA**：支持添加到手机桌面、离线访问、全屏无浏览器 UI 运行
- **数据本地存储**：所有数据存储在本机，不上传任何服务器

- **Service Worker**：版本更新自动提示，一键强制刷新最新版

## 🔒 隐私说明

本工具**不收集任何数据**，所有收藏信息、图片均仅存储在您的设备本地，清除浏览器数据或卸载 App 会同步清除，建议定期导出备份。
